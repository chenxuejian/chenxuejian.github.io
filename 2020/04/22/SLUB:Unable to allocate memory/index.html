SLUB:Unable to Allocate Memory

Created2020-04-22|Updated2021-01-27|Linux
|Post View:

故障

如图：系统日志中报出不能分配内存

解决方法：

  • 临时解决：重启相关节点
  • 永久解决：
    • 升级内核：3.10.0-1062.XXX.el7.x86_64
    • 添加内核参数 cgroup.memory=nokmem
    • 保存退出，刷新grup菜单，并重启节点使永久生效
    • 查看是否生效
